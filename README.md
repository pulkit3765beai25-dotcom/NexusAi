# NexusAI 🚀
NexusAI is a futuristic AI aggregation platform that allows users to access and compare multiple advanced AI models in one place.  
The platform provides side-by-side AI responses in real time using a clean and modern glassmorphism interface.
This project was developed using HTML, CSS, and JavaScript and hosted using GitHub Pages.
## Features
- Multiple AI model integration
- Real-time AI responses
- Side-by-side AI comparison
- Modern glassmorphism UI
- Login & authentication system
- Pricing and subscription interface
- AI image generation support
- Privacy Policy, Terms & Support pages
- Fully responsive design
- Animated futuristic interface
## Supported Models
- GPT-4o
- Claude Sonnet
- DeepSeek R1
- Phi-4
## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript
- OpenRouter API
- GitHub Pages (for hosting)
